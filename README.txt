# 🚀 Projetos do Portfólio

## 🏡 House Rocket – Insights para Maximização de Lucros

Projeto de análise de dados para uma empresa imobiliária com o objetivo de identificar **quais imóveis comprar e qual o melhor momento para vendê-los**.

### Problema de negócio

Responder duas perguntas estratégicas:

1. Quais imóveis comprar entre mais de 20 mil disponíveis?
2. Qual a melhor sazonalidade para vender e por qual preço?

### Solução

Foi construída uma análise exploratória de dados utilizando Python e um **dashboard interativo em Streamlit** para apoiar decisões do time de negócios.

### Ferramentas utilizadas

- Python
- Pandas
- NumPy
- Seaborn
- Streamlit
- Git e GitHub

## 🚲 Cyclistic Bike Share – Análise de Comportamento de Usuários

Projeto de análise de dados para identificar **diferenças de comportamento entre usuários casuais e membros anuais** de um serviço de compartilhamento de bicicletas.

### Objetivo

Gerar insights que ajudem a empresa a **converter usuários casuais em assinantes anuais**.

### Perguntas de negócio

1. Como membros e usuários casuais utilizam as bicicletas?
2. Quais dias e horários apresentam maior uso?
3. Quais oportunidades existem para aumentar a conversão de assinaturas?

### Ferramentas utilizadas

- SQL
- Python
- Pandas
- Tableau
- Git e GitHub
